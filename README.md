# Project Overview: Red-Git Integration Tool

Red-Git is a tool that bridges Redmine and GitLab, enabling seamless searching of issues and related commits across both platforms.

## Key Components
- **Integration**: Connects Redmine and GitLab via their APIs.
- **Search**: Finds issues and commits by project, version, and time range.
- **Display**: Shows tracked (issue-linked) and untracked commits in a table.

## Project Structure
- **Frontend**: Custom UI components for input and data display.
- **Backend**: Handles API communication and server logic.
- **Config**: Uses a JSON file for API tokens and settings.
- **Documentation**: Outlined in `README.md` (this file).

A streamlined solution for cross-platform issue and commit tracking.
